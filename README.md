## 📖 সংক্ষিপ্ত পরিচিতি

**শাড়িঘর (ShariGhor)** হলো বাংলাদেশের নারীদের জন্য একটি প্রিমিয়াম অনলাইন শাড়ির দোকান। এটি একটি **Dummy/Demo eCommerce Project** যা আধুনিক web technology দিয়ে তৈরি। Daraz, Aarong এবং একটি প্রিমিয়াম boutique-এর সমন্বয়ে এর UI/UX ডিজাইন করা হয়েছে।

> 🎯 **Target Audience:** ১৮–৪৫ বছর বয়সী বাংলাদেশি নারী যারা Facebook/Instagram থেকে online shopping-এ অভ্যস্ত।

## 🚀 লোকালি রান করুন

### Prerequisites

নিচের টুলগুলো আগে ইনস্টল করুন:

- [Node.js](https://nodejs.org/) `>= 18.0.0`
- [Git](https://git-scm.com/)
- [Supabase Account](https://supabase.com/) (ফ্রি টায়ার যথেষ্ট)

### ১. Repository Clone করুন

```bash
git clone https://github.com/yourusername/sharighor.git
cd sharighor
```

### ২. Dependencies Install করুন

```bash
npm install
# অথবা
yarn install
# অথবা
pnpm install
```

### ৩. Environment Variables সেটআপ করুন

`.env.local` ফাইল তৈরি করুন:

```bash
cp .env.example .env.local
```

এরপর `.env.local` ফাইলে নিচের values দিন:

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key

# App
NEXT_PUBLIC_APP_URL=http://localhost:3000
NEXT_PUBLIC_APP_NAME=শাড়িঘর

# Payment (Optional for demo)
BKASH_APP_KEY=your_bkash_key
BKASH_APP_SECRET=your_bkash_secret
NAGAD_MERCHANT_ID=your_nagad_merchant_id
```

### ৪. Database সেটআপ করুন

```bash
# Supabase CLI দিয়ে migration রান করুন
npx supabase db push

# অথবা Supabase Dashboard থেকে SQL Editor-এ রান করুন
# /supabase/migrations/init.sql ফাইলটি কপি করে paste করুন
```

### ৫. Development Server চালু করুন

```bash
npm run dev
```

এখন browser-এ যান: **[http://localhost:3000](http://localhost:3000)** 🎉

## 🗄️ ডেটাবেজ স্কিমা

```sql
-- Products Table
CREATE TABLE products (
  id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name        TEXT NOT NULL,         -- বাংলা নাম
  name_en     TEXT,                  -- English name
  slug        TEXT UNIQUE NOT NULL,
  description TEXT,
  price       DECIMAL(10,2) NOT NULL,
  compare_price DECIMAL(10,2),       -- Original price (for discount)
  category    TEXT NOT NULL,         -- silk | cotton | jamdani | party | bridal
  images      TEXT[],               -- Array of image URLs
  colors      JSONB,                -- [{name, hex, stock}]
  in_stock    BOOLEAN DEFAULT true,
  stock_count INT DEFAULT 0,
  rating      DECIMAL(2,1) DEFAULT 0,
  review_count INT DEFAULT 0,
  is_featured BOOLEAN DEFAULT false,
  created_at  TIMESTAMPTZ DEFAULT NOW()
);

-- Orders Table
CREATE TABLE orders (
  id            UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_number  TEXT UNIQUE NOT NULL,
  customer_name TEXT NOT NULL,
  customer_phone TEXT NOT NULL,       -- বাংলাদেশ format: 01XXXXXXXXX
  address       TEXT NOT NULL,
  district      TEXT NOT NULL,
  thana         TEXT NOT NULL,
  items         JSONB NOT NULL,       -- Cart items
  subtotal      DECIMAL(10,2),
  discount      DECIMAL(10,2) DEFAULT 0,
  delivery_fee  DECIMAL(10,2) DEFAULT 60,
  total         DECIMAL(10,2) NOT NULL,
  payment_method TEXT DEFAULT 'cod', -- cod | bkash | nagad
  payment_status TEXT DEFAULT 'pending',
  order_status  TEXT DEFAULT 'pending',
  created_at    TIMESTAMPTZ DEFAULT NOW()
);
```
